# ShulkerV2Beta python src
windows only
do not complain if some features doesn't work 
## Installing:
install [python 3.13 ](https://www.python.org/downloads/release/python-3130/) 
```
git clone https://github.com/Gamersmub/ShulkerV2beta-python
cd ShulkerV2beta-python
pip install -r requirements.txt
python main.pyc
enter any key all will work
```
